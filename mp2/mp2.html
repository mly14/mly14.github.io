MP2 Writeup

Overview and Usage

This project was an exploration of the p5.js Javascript library. This library is great for creating fun visual designs that can be interactive. I created 6 sketches that demonstrate the functionality of this library and compiled them into a gallery.

Development Process

The development process varied from sketch to sketch. In general, I looked for inspiration from the resources Hannah posted on Github. Once I had an idea, then I would begin executing it, oftentimes borrowing starter code from examples online. I coded all of my sketches using the p5.js web editor then copied them into VSCode. The web editor was really easy to use and convenient. I won't go in-depth into the process for all my sketches but below is how I created my first sketch. 

  1. I drew a square.
  2. I drew more squares inside of it by using a recursive function. The terminating condition is the number of levels (the number of squares inside).
  3. I wanted to make the inside squares rotate. I added the rotate() function but it wasn’t performing how I anticipated. The squares were not rotating around the center. 
  4. After reading an online resource, I realized the rotate function rotates around the origin which is (0,0) by default. I used the translate() function to move the origin to the center of the screen. This fixed my problem and now the squares are correctly rotating inside the main square.
  5. I increased the square size, added more levels, and added a black border to make the design pop out. 
  6. I had to fiddle around with the sizing to make it responsive to the window width and height.

Issue Deep-Dive

My first attempt at making the gallery page was to use a CSS grid. I ended up using a different method to display my sketches (iframes) but this was still a problem I encountered. First, I copied the code for a CSS grid from Hannah's site and was trying to put an image into the div. The image wasn’t showing up for some reason. I searched online for answers and came across a StackOverflow post with the same situation. They suggested styling the image width to be 100% I thought this made sense because the image would take the size of the div. However, my problem still persisted. Then I looked more closely at my code and realized padding was applied to the divs. I removed it and the image was able to be displayed. I guess the padding was taking up all the space preventing the image from being displayed. I learned that sometimes you need to review your code and understand it before Googling answers.
