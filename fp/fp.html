438 Final Project Proposal

General Description

For my final project, I am making a game that will use Firebase to implement a user login flow and backend database that stores high scores.

Sketches

Descriptions

Prototype

For my prototype, I will have the login, fundamentals of the game, and backend storing high scores working. The main parts of my game will be the rising platforms and the rolling ball. The ball will be controlled with the arrow keys. 

Final Version

My final version will have more interesting styling such as adding a face to the ball. I will add a fun background. I will also make the game more challenging by gradually increasing the speed of the platform. To make it more fun, I will add power-ups. If you collect a power-up you can go through the platforms for a certain amount of time. 

Development Process

Prototype Version

  1. Make the game with the rising platforms and rolling ball. 
  2. Add a starting screen.
  3. Add a game over screen with a replay button.
  4. Set up the firebase database.
  5. Add a login screen to the game and have information stored in the database
  6. Add a high scores to the game over screen that displays the top 5 players. 

Final Version

  1. Add stylings to the game to make it more visually appealing. This will include changing font styles and adding color. I may also add animation to the ball.
  2. Add some difficulty to the game by gradually increasing the speed of the platforms.

A list of known unknowns

I don’t know much about Firebase but from Hannah’s tutorial and the resources she’s provided, I will learn how to use it to implement a database storing users and high scores. I will need to figure out to get my game to display data from the database. I didn’t use p5 Play for the previous assignment so I’ll need to read through the documentation and familiarize myself with the methods.

Stretch goals.

Most of my stretch goals are about making the game more visually appealing and interesting.

  • If I have time I will change the face on the ball. Once it gets closer to the top, it will turn into a sad face.
  • I want to add “power-ups” so when the ball collects it, the platforms will move out of the way 
  • I will also have buttons for starting and replaying the game instead of just clicking anywhere on the screen.
  • I will add music and sound effects.
